## Description API-SHOP

Api servicios de productos de inventario

## Installation

```bash
$ yarn install
```
## Descargar base de datos POSTGRESQL PARA DOCKER

```bash
$ docker pull postgres
```

## Levantar la base de datos POSTGRESQL

```bash
$ docker-compose up -d
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
